# 🌐 SEO Blog Website

A fully responsive SEO-focused blog website built using **HTML, CSS, and Vanilla JavaScript**.  
This project demonstrates practical implementation of **basic frontend development + basic SEO concepts**.

---

## 🚀 Features

- Responsive design (mobile, tablet, desktop)
- Sidebar navigation with hamburger menu
- Blog listing page with multiple articles
- Clean UI with proper spacing and layout
- Contact form with JavaScript interaction
- Smooth animations using AOS
- Typing effect using Typed.js
- Dark mode toggle button

---

## 🧠 SEO Features

- Semantic HTML structure
- Title tags & meta descriptions
- Internal linking between pages
- `robots.txt` for search engine crawling
- `sitemap.xml` for indexing
- Structured content for better readability

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- AOS (Animate on Scroll)  
- Typed.js
